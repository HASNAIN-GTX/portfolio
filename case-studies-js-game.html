Case Study: JavaScript Game

Role: Developer
Stack: JavaScript, Canvas API
Result: 60 FPS on mobile, 100% Lighthouse PWA score

Overview

This project is a browser-based game built entirely with JavaScript and the Canvas API. The goal was to demonstrate game development fundamentals, deliver smooth performance on mobile devices, and utilize progressive web app (PWA) technologies.

Goals

  • Build a fun, interactive browser game playable on desktop and mobile
  • Achieve smooth 60 FPS animation and gameplay, even on low-end devices
  • Implement PWA features for offline play and installability
  • Score 100% on Lighthouse performance, accessibility, and PWA audits

Challenges

  • Rendering and game logic optimization for mobile devices
  • Efficient asset management with limited resources
  • Ensuring accessibility for different user input types (touch, keyboard)

Solutions

  • Used requestAnimationFrame and optimized draw calls for smooth performance
  • Minimized asset sizes and deferred non-critical scripts
  • Implemented responsive controls (touch and keyboard)
  • Configured service worker for offline caching and PWA compliance

Impact

  • Consistent 60 FPS on mobile and desktop
  • 100% Lighthouse score for PWA, performance, and accessibility
  • Positive feedback from users on responsiveness and ease of play
